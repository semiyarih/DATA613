  • stat myfile.txt The name of the file.
  • Size: 0 The size of the file in bytes.
  • Blocks: 0 The number of blocks for the file.
  • IO Block: 65536 The size of each block.
  • regular empty file Indicates if the file is egular file, directory, etc
  • Device: d0ab7935h/3500898613d The device number.
  • Inode: 101612466592571689The inode number (file serial number)
  • Links: 1 he number of links to the file.
  • Access: (0644/-rw-r--r--) The access permissions for the file.
  • Uid: (1754655/semiyari) The user ID of the file’s owner.
  • Gid: (1049089/ UNKNOWN) The group ID of the file’s group.
  • Access: 2024-07-08 17:36:31.153073900 -0400 The last access time.
  • Modify: 2024-07-08 17:36:31.153073900 -0400 The last modification time.
  • Change: 2024-07-08 17:36:31.152213300 -0400 The last status change time.
  • Birth: 2024-07-08 17:36:31.152213300 -0400 The creation time (if supported by the file system).ls
